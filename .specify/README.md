This project uses Spec-Driven Development (SDD).

## Directory Structure
*   `.specify/memory/constitution.md` - The project's source of truth and guidelines.
*   `specs/` - Feature specifications.
    *   `specs/mvp/1_initial_features.md` - MVP Requirements.

## How to use
When asking the AI to implement features, always refer to the relevant spec in the `specs/` folder.
The AI should cross-reference `.specify/memory/constitution.md` for architectural rules.
